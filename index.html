Reciprocity Game

Thank you for participating in this experiment.

This experiment uses a 'Reciprocity Game' and takes approximately 10 minutes to complete.

In the Reciprocity Game, two players take turns choosing to Cooperate or Defect to earn rewards.

Game Flow Overview

The diagram below shows the overall game process.

※ This diagram is an example to help you understand the game structure.

• The dark P1 represents Player 1 (You).

• The light P2 represents Player 2 (Opponent).

• Turns alternate in the order: P1 → P2 → P1.

What Each Choice Means

※ This diagram is an example to help you understand the game structure.

• On each turn, a player can choose Cooperate (C) or Defect (D).

• Cooperate (C) = the horizontal arrow → (the game continues to the next player).

• Defect (D) = the downward arrow ↓ (the game ends immediately with the payoff shown below).

Understanding Rewards

※ This diagram is an example to help you understand the game structure.

• The star symbols (★) in the diagram represent rewards.

• ★ (dark stars) = Your reward.

• ☆ (light stars) = Opponent's reward.

• Players receive different rewards depending on both players' choices.

Key Insight: Cooperation vs. Defection

※ This diagram is an example to help you understand the game structure.

• If all players cooperate, both players earn a moderate reward (★★ / ☆☆).

• If you defect at the final turn (Turn 3), you take maximum reward (★★★★) but your opponent gets nothing.

• If you defect first (Turn 1), the game ends early with a small reward for both (★ / ☆).
